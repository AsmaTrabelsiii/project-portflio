Projects

Project Name: BizAnalytics – Full-Stack BI Dashboard

A minimalist, data-driven Business Intelligence dashboard designed for e-commerce store managers. It aggregates sales data, tracks inventory levels, and visualizes monthly revenue growth through interactive charts. Implemented dynamic filtering by date and product category to provide actionable insights. Tech Stack: React.js, Node.js, Express, MongoDB, Chart.js / Recharts, Tailwind CSS. Key Features: Real-time data visualization with clean, interactive charts. Advanced data filtering and sorting capabilities. Responsive, minimalist layout designed for optimal desktop and tablet view.

PrintFlow – Document Ordering & Management Platform

A full-stack web application that simplifies document print-ordering and formatting workflows. Users can upload documents, customize print preferences (paper size, color options), and receive instant automated cost calculations. Features a comprehensive admin panel to track order lifecycles and manage customer queues. Tech Stack: React.js, Context API, Node.js, Express, MongoDB (GridFS for file storage), Tailwind CSS. Key Features: Secure file upload and metadata processing. Automated dynamic pricing engine based on document properties. Admin dashboard for real-time status updates and order tracking..

Taskify – Agile Project Management Tool

A highly interactive project management application built to help teams organize tasks visually. It features dynamic kanban boards with seamless drag-and-drop functionality for updating task statuses instantly. Optimized for performance and smooth UI transitions. Tech Stack: React.js, React Beautiful DND, Node.js, Express, MongoDB. Key Features: Smooth drag-and-drop workflow for lists and individual task cards. User authentication and personalized dashboard management. Clean, distraction-free user interface focused on productivity.
